# bc2025_app — 筋電ダーツゲーム

BITalino の筋電（EMG）センサーと加速度センサーを使って、腕の筋電でダーツを投げる体感型ゲームです。

## ゲームの概要

腕に装着した筋電センサーに力を入れると閾値を超えた瞬間にダーツが発射されます。加速度センサーの値が投げる力（飛距離）に反映されます。揺れる腕の角度に合わせてタイミングよく投げ、障害物を避けて的に当てればクリアです。

| 項目 | 内容 |
|---|---|
| ステージ数 | 3 |
| 発射トリガー | 筋電（チャンネル A3）が閾値 650 を超えた瞬間 |
| 投げる力 | 加速度（チャンネル A4）の平均値から算出 |
| 障害物 | サンゴ・テトラポッド |

## 必要環境

- Python 3.8+
- BITalino デバイス（EMG: チャンネル A3、加速度: チャンネル A4）

## インストール

```bash
pip install pygame bitalino
```

## 使い方

```bash
python final_darts.py
```

1. タイトル画面でクリックまたはスペースキーを押す
2. ステージ番号を選択
3. 腕に BITalino を装着し、タイミングを合わせて力を入れてダーツを投げる
4. 的に当てるとクリア、リトライボタンでやり直し可能

## 設定の変更

`final_darts.py` 上部の定数を調整できます：

```python
macAddress = "98:D3:11:FE:02:2B"  # BITalino の MAC アドレス
tap_threshold = 650                 # ダーツ発射の筋電閾値
```
